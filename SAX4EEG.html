First studies of EEG signals¶

In order to do some initial analysis of EEG signals we download the UCI EEG Eye State Data Set.

  • Dataset obtained from a continuous EEG measurement.
  • EEG technology used: Emotiv EEG Neuroheadset.
  • Duration: 117 seconds.
  • Last column: Eye state detected using a camera and added manually. (1: eye-closed, 0: eye-opened)
  • Values in chronological order (from the top to the bottom of the dataset)
  • Since the dataset contains 14980 points, the frequency of measures is of 128 points per second, which is close to 10 times the frequency of Beta wave.

Filtering outliers¶

The EEG records contain some unusually high values. It is better to filter these measures in order to have a better repartition of the data points after the z-score standardization

In [4]:
def clean_outlier(column):
    mean_column = column.mean()
    std_column = column.std()
    column[column > mean_column+3*std_column] = mean_column+3*std_column
    column[column < mean_column-3*std_column] = mean_column-3*std_column
    return column

Compute the Piecewise Aggregate Approximation (PAA)¶

The PAA algorithm applies to the time series the following operations:

+ Segments the time series into equal parts 
+ Stores the average value for each part

Compute the Symbolic Aggregate approXimation (SAX)¶

The SAX algorithm applies to the time series the following operations:

+ Applies the PAA algorithm 
+ Converts the PAA results into symbols
